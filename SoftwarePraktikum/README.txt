GENERELLE NUTZUNGSHINWEISE
----------------------------------------------------------
- käpsele.de benötigt für den reibungslosen Betrieb eine
  installierte Java Runtime Environment 1.7 oder höher
- käpsele.de benötigt für die reibungslose Darstellung
  Firefox 39, Chrome 42, Internet Explorer 11 oder jeweils
  höher
  
  
INSTALLATIONSHINWEISE
----------------------------------------------------------
- Bitte beachten Sie dafür die Datei DB_Install_Anleitung
- Die Klasse Beispieldaten füllt die Datenbank mit Dummies
